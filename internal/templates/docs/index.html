Nano Crypto Games

Real-money browser games powered by Go and Nano (XNO) cryptocurrency — feeless, instant, and built for microtransactions.

Go 1.25+ Ӿ Nano (XNO) PostgreSQL 16
Getting Started

Requirements

  • Go 1.25+ — the server and all tooling are written in Go
  • Docker — used to run a local Postgres instance
Getting Started

Local Development

1 Start Postgres

docker run -d \
  --name nano_crypto_games_local_db_container \
  -e POSTGRES_USER=nano \
  -e POSTGRES_PASSWORD=nano \
  -e POSTGRES_DB=nano_crypto_games_local_db \
  -p 5432:5432 \
  postgres:16

2 Build and run the server

go build ./...
go run ./cmd/server

3 Stop and remove the container

docker stop nano_crypto_games_local_db_container
docker rm nano_crypto_games_local_db_container
Configuration

Environment Variables

Copy env-example.txt to .env and fill in your values. Variables without a default are required for the relevant feature to work.

Variable Description Default
DATABASE_URL Postgres connection string postgres://nano:nano@localhost:5432/nano_crypto_games_local_db?sslmode=disable
DONATION_ADDRESS Nano address to receive donations —
FAUCET_SEED Seed for the faucet wallet —
FAUCET_TEST_MODE Disable real payouts for testing false
NANO_MASTER_SEED HD wallet master seed — keep secret —
NANO_RPC_API_KEY API key for the primary Nano node —
NANO_RPC_FALLBACK_URL Fallback public Nano node https://us-1.nano.to
NANO_RPC_PRIMARY_URL Primary public Nano node https://rpc.nano.to
Configuration

Master Seed

A 32-byte secret used to derive every player's Nano wallet via HD derivation.

⚠
Back it up. Losing the master seed means losing access to all funds held in player wallets. Never commit it or reuse it across environments — if you change it, all existing wallet addresses will change.

Generate one with OpenSSL:

openssl rand -hex 32

Put the 64-character hex output in your .env as NANO_MASTER_SEED.

ℹ
If NANO_MASTER_SEED is not set, the server generates a random ephemeral seed on each start — wallets are non-persistent in this mode, suitable only for local development.
